PDF Tool WinForms App

Giải pháp tối ưu để đếm trang, gộp file, đổi tên hàng loạt và nén file PDF. Đơn giản, mạnh mẽ và hiệu quả cho công việc văn phòng.

Tải xuống cho Windows

Các chức năng chính

Tất cả công cụ bạn cần để quản lý tài liệu PDF một cách chuyên nghiệp.

Đếm số trang PDF

Quét toàn bộ file PDF trong thư mục (bao gồm thư mục con) và xuất ra file Excel với thông tin chi tiết.

Gộp PDF

Tự động tìm và gộp toàn bộ file PDF trong các thư mục được chỉ định thành một file duy nhất `FileTong.pdf`.

Đổi tên sau ký số

Tự động hóa việc sắp xếp lại file và thư mục sau khi ký số, dọn dẹp thư mục `Unknown`.

Nén file PDF >100MB

Sử dụng Ghostscript để nén các file PDF dung lượng lớn mà vẫn đảm bảo chất lượng ở mức tốt nhất.

Hướng dẫn sử dụng

Chỉ với 4 bước đơn giản để hoàn thành công việc của bạn.

1

Mở App

Mở file `PdfToolWinFormsApp.exe`

2

Chọn thư mục

Chọn thư mục gốc chứa các file PDF.

3

Chọn chức năng

Chọn 1 trong 4 chức năng có sẵn.

4

Nhấn "Chạy"

Theo dõi tiến trình và nhận kết quả.

Yêu cầu hệ thống

Để đảm bảo phần mềm hoạt động ổn định, máy tính của bạn cần đáp ứng các yêu cầu sau:

Hệ điều hành

Windows 10/11 (64-bit)

.NET Framework

4.8 trở lên

Ghostscript (Bắt buộc cho chức năng nén)

Cần cài đặt sẵn. Ví dụ: `C:\Program Files\gs\gs10.03.0\bin\gswin64c.exe`

Bắt đầu ngay hôm nay!
